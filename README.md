## Quick Start

### For Students:
1. Visit the website
2. Browse symbols by category or use search
3. Click any symbol to copy instantly
4. Use the Text Composer to build complex expressions
5. Hold symbols to add to favorites for quick access

### For Teachers:
1. Open the Text Composer at the top
2. Type or paste text directly
3. Click symbols to add them to your composition
4. Copy the entire text with one click
5. Use in lesson plans, worksheets, or presentations
